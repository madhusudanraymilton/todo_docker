ZKBio Time API Integration

Connect your Odoo system with ZKBio Time biometric devices to automate attendance tracking and management.

Features

API Configuration

Easy setup and configuration of ZKBio Time API connection with token-based authentication.

Terminal Management

Sync and manage all your biometric terminals from a single interface.

Real-time Status

Monitor online/offline status of your biometric devices in real-time.

Attendance Sync

Automatically sync attendance data from biometric devices to Odoo HR Attendance.

Installation

  1. Copy the module to your Odoo addons directory
  2. Update the apps list
  3. Install the "ZKBio Time API" module
  4. Configure your API settings in Configuration menu

Configuration

  1. Go to ZKBio Time > Configuration > API Configuration
  2. Create a new configuration
  3. Enter your API URL (e.g., http://103.91.230.83:8088)
  4. Enter your API authentication token
  5. Click "Test Connection" to verify
  6. Click "Sync Terminals" to import your devices

Usage

Once configured, navigate to ZKBio Time > Terminals > Biometric Terminals to view and manage your devices.
